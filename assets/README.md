# Assets

This folder contains images and media for the documentation.

## Required Images

To complete the README, add the following images:

- `banner.png` - Header banner image (recommended: 1200x400px)
- `workflow-demo.gif` - Animated GIF showing the workflow in action
- `workflow-example.png` - Screenshot showing the correct node setup (Load Image → BespokeAI 3D Generation → 3D Preview)
- `quickstart-workflow.png` - Screenshot of a basic workflow setup
- `example-basic.png` - Screenshot of basic image-to-3D result

## Recommended Image Specifications

| Image | Size | Format | Description |
|-------|------|--------|-------------|
| banner.png | 1200x400 | PNG | BespokeAI branding + "3D Generation for ComfyUI" |
| workflow-demo.gif | 800x450 | GIF | Screen recording of a complete generation |
| workflow-example.png | 1920x1080 | PNG | Full workflow screenshot with all nodes connected |
| quickstart-workflow.png | 800x400 | PNG | ComfyUI workflow screenshot |
| example-basic.png | 1000x500 | PNG | Before/after with 3D model preview |
